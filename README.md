## About the Project

**Video2Blog** is an automated AI pipeline designed to seamlessly transform video content into well-structured, SEO-friendly blog posts. By extracting audio, generating highly accurate transcripts, and leveraging Large Language Models (LLMs), this tool synthesizes spoken content into readable articles complete with headings, bullet points, and summaries.

Whether you are a content creator looking to repurpose your YouTube videos or a developer exploring audio-to-text AI pipelines, Video2Blog provides a modular, scalable architecture for rapid content generation.

### Core Features
* **Automated Extraction:** Seamlessly downloads video/audio payloads using robust extractors.
* **Intelligent Transcription:** Utilizes advanced speech-to-text models (like OpenAI's Whisper) for high-accuracy transcriptions.
* **Contextual Formatting:** Prompts an LLM to rewrite transcripts into engaging blog formats, stripping out filler words and structuring the narrative.
* **Ready-to-Publish Export:** Outputs clean `.md` files that can be instantly pasted into WordPress, Ghost, or static site generators.

## Prerequisites

* Python 3.9+
* FFmpeg (Required for local audio extraction and processing)
* API Keys for your chosen LLM provider (e.g., OpenAI, Anthropic, or Groq)
